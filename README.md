## Setup

1. Clone the project

```bash
    git clone https://github.com/Thuch-Mager/imag
```

2. Navigate to the project

```bash
    cd amal
```

3. install dependencies

```bash
    npn install
```
4. Run project locally
  ```bash
      npn npm start
  ```
